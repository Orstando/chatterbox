## How to Run the Server ##

### Running Chatterbox ###

```bash
git clone https://github.com/Orstando/chatterbox.git
cd chatterbox

# Install dependencies
npm i

# Setup server configuration
cp config.example.js config.js
# Now would be a great time to edit config.js

node .
```
